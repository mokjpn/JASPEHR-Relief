# JASPEHR-Relief

## 日本語
このリポジトリは、[JASPEHRプロジェクト](https://jaspehr.jp/) のファンのために、プロジェクトのロゴを3Dプリントできるよう、STLファイルにしたものを格納しています。作成は [OpenSCAD](https://openscad.org/) を使っているので、OpenSCADのソースコードも置きました。楽しんで！

## English
This repository contains STL files for 3D printing the JASPEHR Project logo for fans of the [JASPEHR Project](https://jaspehr.jp/). The design is created using [OpenSCAD](https://openscad.org/), and the OpenSCAD source code is included in this repository. Enjoy!
